# Carlos Artur Guimaraes - Portfolio v3

Portfolio profissional estatico em HTML e CSS, com foco em educacao, dados, politicas publicas e cooperacao internacional.

## Visao geral

Este projeto apresenta:

- perfil profissional
- trajetoria institucional
- pilares de atuacao
- cases selecionados
- producao academica e tecnica
- formacao
- contato

Tambem inclui um link para visualizacao sistemica no Retina sobre conexoes entre autores e eixos STEAM.

## Estrutura

- `index.html`: pagina principal do portfolio
- `styles.css`: estilos e responsividade

## Publicacao

Este projeto esta preparado para publicacao via GitHub Pages.

Repositorio:

- [tucaguimaraes/carlosguimaraes](https://github.com/tucaguimaraes/carlosguimaraes)

URL publica esperada:

- [https://tucaguimaraes.github.io/carlosguimaraes/](https://tucaguimaraes.github.io/carlosguimaraes/)

Para publicar pelo GitHub Pages:

1. Abra `Settings` no repositorio.
2. Va em `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch `main` e a pasta `/(root)`.
5. Salve e aguarde a publicacao.

## Edicao local

Para visualizar localmente, basta abrir o arquivo `index.html` no navegador.

Se quiser subir novas alteracoes:

```bash
git add .
git commit -m "Atualiza portfolio"
git push
```
